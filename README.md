# Изменения в проекте Yellow Land

## 1. Кнопка полёта на домиках
- Добавлена синяя сфера-кнопка на крышу каждого дома в деревне
- Управление: WASD для движения, W/S для вертикального перемещения

## 2. Запрет спавна объектов на пути оранжевого NPC
- Увеличена зона запрета спавна до 15 единиц вокруг маршрута патрулирования
- Оранжевый NPC патрулирует по кругу радиусом (islandSize/2 - 10)
- Горы и деревья не спавнятся на пути NPC

## 3. Скорость оранжевого NPC
- Скорость патрулирования понижена до 0.00015 (было 0.0005)
- Примерно 0.30 от исходной скорости

## 4. Диалоговая система с оранжевым NPC
- При взаимодействии (E) NPC останавливается
- Открывается диалоговое окно с выбором реплик
- Стартовые фразы (случайно):
  - "Привет! Я фанат новой земли!"
  - "Привет, как дела?"
  - "Привет!"
- Древовидная структура диалога с множеством вариантов ответов
- Темы диалогов: о себе, об острове, о других NPC, о новых землях
- После завершения диалога NPC продолжает патрулирование
- Диалог закрывается автоматически через 2 секунды после последней фразы

## 5. Коллизии NPC с объектами
- Все NPC (оранжевый, фиолетовый, зелёный, розовый) не проходят сквозь препятствия
- Проверка коллизий с горами, деревьями, домами и другими NPC
- При столкновении NPC меняет направление или остаётся на месте
- Функция выталкивания NPC из препятствий если застрял

## 6. Мяч
- Создан белый мяч (сфера) в мире
- Подбор: клавиша E, когда мяч рядом (5 единиц)
- Бросок: при нажатии E с мячом в руке
- Мяч летит в противоположную сторону от взгляда игрока
- Физика: гравитация, отскоки от земли
- Коллизии мяча с препятствиями (горы, деревья, дома) - отскок
- При попадании в NPC:
  - NPC подлетает вверх (сила 0.6)
  - Падает на землю
  - Лежит ~1 секунду (состояние 'hit')
  - Встаёт и продолжает свои дела
- Добавлены подсказки:
  - Зелёная: "Нажми E чтобы взять мяч" (когда мяч рядом)
  - Оранжевая: "Нажми E чтобы бросить мяч" (когда мяч в руке)

## 7. Платформа телепортации
- Красная кнопка на серой стенке в деревне (позиция -25, -35)
- При нажатии E появляется железная платформа на высоте 80 единиц
- Размер платформы: 40x40 единиц (увеличено в 2 раза)
- Игрок телепортируется на платформу
- Можно свободно ходить по платформе
- Можно спрыгнуть с платформы в любой момент
- При падении ниже 80 единиц платформа исчезает
- Добавлены индикаторы:
  - Фиолетовый: "ПЛАТФОРМА АКТИВНА"
  - Подсказка: "Нажми E для телепортации" (когда рядом с кнопкой)

## Изменённые файлы
- main.js - основная логика игры
- index.html - добавлены UI элементы (индикаторы, диалоговое окно, подсказки)
- style.css - стили для новых UI элементов

## Новые UI элементы
- #fly-indicator - индикатор режима полёта
- #platform-indicator - индикатор платформы
- #teleport-prompt - подсказка телепортации
- #ball-pickup-prompt - подсказка подбора мяча
- #ball-throw-prompt - подсказка броска мяча
- #dialog-window - диалоговое окно с NPC

## Новые функции
- createFlyButton() - создание кнопки полёта на доме
- createVillageTeleportButton() - создание кнопки телепортации
- createPlatform() - создание платформы
- activatePlatform() / deactivatePlatform() - управление платформой
- createBall() - создание мяча
- pickUpBall() / throwBall() - подбор и бросок мяча
- updateBall() - физика мяча
- hitNPC() - попадание мяча в NPC
- startDialog() / showDialogNode() / closeDialog() - диалоговая система
- checkObstacleCollision() - проверка коллизий NPC
- pushNpcOutOfObstacles() - выталкивание NPC из препятствий
