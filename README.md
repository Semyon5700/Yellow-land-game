# Список изменений в проекте Yellow Land game!

================================================================================
## 1. ИСПРАВЛЕНИЕ КОЛЛИЗИИ СТЕНКИ С КНОПКОЙ ТЕЛЕПОРТАЦИИ
================================================================================
- Добавлена коллизия для стенки с кнопкой телепортации
- Стенка добавлена в массив obstacles с радиусом 2.5 и типом 'wall'
- Файл: main.js (функция createVillageTeleportButton)

================================================================================
## 2. ДОБАВЛЕНО ГЛАВНОЕ МЕНЮ
================================================================================
- Создано главное меню с фоновым изображением Menu.png
- Добавлена надпись "Yellow Land" жёлтым цветом в левом верхнем углу
- Добавлены кнопки: "Играть", "Сменить режим графики"
- Добавлен текст авторства: "2026 год автор Semyon5700 игра Yellow land game"
- Файлы: index.html, style.css, main.js

================================================================================
## 3. ИЗМЕНЕНИЕ ОКНА ВЫБОРА ГРАФИКИ
================================================================================
- Окно выбора графики сделано компактным (модальное окно)
- Добавлен фон Menu.png с затемнением
- Файлы: index.html, style.css

================================================================================
## 4. КНОПКА ВЫХОДА В ГЛАВНОЕ МЕНЮ
================================================================================
- Добавлена кнопка "Выйти в главное меню" в меню паузы
- Реализована навигация между меню паузы и главным меню
- Файлы: index.html, main.js

================================================================================
## 5. ЗАМЕНА ТЕЛЕПОРТАЦИИ ЧЕРЕЗ КНОПКУ НА ТЕЛЕПОРТАЦИЮ ЧЕРЕЗ NPC
================================================================================
- Удалена кнопка телепортации со стенкой в деревне
- Удалена функция createVillageTeleportButton()
- Удалены переменные: villageTeleportButton, playerOnPlatform, teleportPrompt
- Добавлена опция "Телепортируй меня на платформу!" в диалог с оранжевым NPC
- Добавлена ветка диалога teleport
- Телепортация вызывается через функцию activatePlatform() при выборе опции
- Файлы: main.js, index.html, style.css

================================================================================
## 6. ИСПРАВЛЕНИЕ ФОНА ОКНА ВЫБОРА ГРАФИКИ
================================================================================
- #graphics-selector теперь использует Menu.png как фон
- Добавлен полупрозрачный затемняющий слой (::before)
- Файл: style.css

================================================================================
## 7. ДОБАВЛЕНИЕ ЗАГОЛОВКА В ГЛАВНОМ МЕНЮ
================================================================================
- Изменено: "Yellow Land game"
- Файл: index.html

================================================================================
## 8. ОПТИМИЗАЦИЯ И ЧИСТКА КОДА
================================================================================
Удалён неиспользуемый код:
- Функция showNPCMessage() (нигде не вызывалась)
- Функция updateNPCMessagePosition() (обновляла несуществующий элемент)
- Переменные: npcMessageElement, canShowMessage, fpsCounter
- Элемент <div id="npc-message"> из HTML
- CSS-стили для #npc-message

================================================================================
## 9. ИСПРАВЛЕНИЕ ПАДЕНИЯ FPS НА ПЛАТФОРМЕ
================================================================================
- Добавлена проверка коллизий с препятствиями при движении на платформе
- checkNPCCollisions() вызывается только когда игрок на земле
  (не в полёте и не на платформе)
- Удалено playerOnPlatform = false из deactivatePlatform()
- Файл: main.js

================================================================================
## 10. УДАЛЕНИЕ РЕЖИМА ПОЛЁТА
================================================================================
Полностью удалён режим полёта:
- Удалена функция createFlyButton() (голубые сферы на крышах домов)
- Удалена функция activateFlyMode()
- Удалены переменные: flyButtons[], flyButtonEnabled, flyIndicator, 
  flyMode, spacePressCount, lastSpaceTime
- Удалён код режима полёта из updatePlayer()
- Удалена обработка клавиши F и двойного нажатия пробела
- Удалён HTML-элемент #fly-indicator
- Удалены CSS-стили для индикатора полёта
- Файлы: main.js, index.html, style.css
