# Yellow Land game

3D-игра от первого лица на Three.js. Открытый мир, NPC, физика мяча, диалоги и телепортация на летающую платформу.

| Клавиша | Действие |
|---------|----------|
| `WASD` | Перемещение |
| `Мышь` | Вращение камеры |
| `Пробел` | Прыжок |
| `E` или `ПКМ` | Взаимодействие (диалог, подобрать/бросить мяч) |
| `Esc` | Пауза / выход из диалога |

## Игровой процесс

- Исследуйте остров с деревней, горами и лесами
- Общайтесь с 4 NPC через диалоговое окно
- Попросите оранжевого NPC телепортировать вас на платформу (высота 80)
- Найдите белый мяч, подберите (E) и бросайте в NPC — они временно падают
- Настройте графику: низкая / средняя / высокая (тени, сглаживание)

## Запуск

**Локальный запуск:**  
Скачайте репозиторий и откройте `index.html` в браузере.


##Системные требования

Любой современный браузер с поддержкой WebGL (Chrome, Firefox, Edge).
