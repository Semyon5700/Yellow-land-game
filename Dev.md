## Архитектура проекта (кратко)

| Файл | Назначение |
|------|------------|
| `index.html` | Разметка UI (меню, диалоги, индикаторы) |
| `style.css` | Стили всех окон и элементов |
| `main.js` | Вся игровая логика (3D, физика, NPC, диалоги) |
| `three.min.js` | Библиотека 3D-рендеринга |

## Основные функции в main.js

- `createHumanModel()` — генерация NPC и игрока
- `updatePlayer()` — движение, камера, коллизии
- `updateOrangeNPC()` — патрулирование по кругу
- `updateJumpingNPC()` — логика прыгающих NPC
- `updateBall()` — физика мяча
- `startDialog()` / `showDialogNode()` — диалоговая система
- `applyGraphicsSettings()` — смена режимов графики
- `activatePlatform()` / `deactivatePlatform()` — платформа

## Как добавить новую ветку диалога

1. Добавить объект в `dialogTree`
2. Сослаться на него через `next` в существующей ветке
